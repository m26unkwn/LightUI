Toast

Toast is use to show notification and provide some message to user. You can use it as snackbar too.

Basic Toast

Basic Toast is having a message and a remove button. You can wrap it any container show that give it position.

You can create your Toast using class Toast followed by class toast-success toast-failure and toast-ongoing .
