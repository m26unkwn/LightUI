Image

Images can be responsive to fit the parent's width, and also can be customised to be round shaped.

Round Image

Round Images are the circular image.

You can create your circular image using class round-img.

Responsive Image

Responsive Images is the one that expands to cover the container it is present in.

You can create your responsive image using class img-responsive.


NOTE : Check responsiveness by expanding width.
